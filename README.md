# Corso Python 2026 di AIM

### Parte 1 - Come usare il materiale

Nelle cartelle `lezione-1`, `lezione-2` e `lezione-3` (con relativi `homework-{1, 2, 3}`) si trovano i Python notebook delle lezioni (i files che finiscono per `.ipynb`).

Possono essere comodamente utilizzati tramite [Google Colab](https://colab.research.google.com/), caricandoli o dal vostro PC o tramite URL del notebook: per esempio copiate `https://github.com/AIM-mate/corso-python-26/blob/master/lezione-1/lezione-1.ipynb` nella sezione "GitHub" nella barra a sinistra che si apre da menù "File > Carica Notebook".
