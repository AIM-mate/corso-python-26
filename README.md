# Corso Python 2026 di AIM

### Parte 1 - Come usare il materiale

Nelle cartelle `lezione-1`, `lezione-2` e `lezione-3` (con relativi `homework-{1, 2, 3}`) si trovano i Python notebook delle lezioni (i files che finiscono per `.ipynb`).

Possono essere comodamente utilizzati tramite [Google Colab](https://colab.research.google.com/), caricandoli o dal vostro PC o tramite URL del notebook: per esempio copiate `https://github.com/AIM-mate/corso-python-26/blob/master/lezione-1/lezione-1.ipynb` nella sezione "GitHub" nella barra a sinistra che si apre da menù "File > Carica Notebook".

### Parte 2 - Come usare il materiale

#### Opzione 1: Ambiente di sviluppo online
Una macchina virtuale di Google https://shell.cloud.google.com/

#### Opzione 2: Ambiente di sviluppo online
Il cloud e la maggior parte di server funzionano con Linux, quindi ci allineamo:
- su MacOS tutto ok (Linux e MacOS sono fratellastri)
- su Windows, usiamo una virtual machine: seguite le [istruzioni di installazione di WSL](https://learn.microsoft.com/en-us/windows/wsl/install) e lanciate il comando `wsl --install Ubuntu-24.04` da [Terminale](https://learn.microsoft.com/en-us/windows/terminal/install).

**Editor (*cioè dove scriviamo il codice*)**: installate [VS Code](https://code.visualstudio.com/download)
